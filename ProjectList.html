Responsibilities: I was responsible for the following activities

Maintenance and customization of Glance Salesforce application components such as users, roles, profiles, groups, record types, sharing rules, pick lists and page layout customization to support vital business functions

Creation of validation rules, workflow rules, process builder, flows

Reports and Dashboard creation and Assisted users with report design and management

Used field level security along with page layouts in Lightning to manage access to certain fields

Data load activities using Salesforce data loader application and import wizard

Configuring 3rd party applications on Salesforce from Appexchange

Involved in daily standup meetings, Scrum.
